One-Time Firebase Setup (5 minutes, free)

Do this once and all your product edits will go live instantly for every visitor worldwide.

1
Go to console.firebase.google.com and create a free project
2
In the project: Build → Firestore Database → Create → Start in test mode
3
Build → Storage → Get started → Start in test mode
4
Project Settings → Your apps → Web app → Copy the config
5
Open firebase-config.js and paste your values. Also set your ADMIN_PASSWORD and WHATSAPP_NUMBER
6
Deploy to Vercel. Come back here, log in, and start uploading images and editing products!
